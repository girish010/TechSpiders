Manual Testing Course or Training For Software Test Engineer

What You Will Learn From This Course ?

Documentation of defect report using defect template and defect tracking tools.Knowing How Defect Life Cycle Works. Agile Methodologies, Finally Working with Realtime Project for Manual testing.

           Manual Testing Course duration will be 40Hrs .
